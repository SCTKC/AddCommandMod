# AddCommandMod

/ban プレイヤー名 で名前がプレイヤー名のプレイヤーをbanすることができます。

同じように/kick プレイヤー名 で名前がプレイヤー名のプレイヤーをkickすることができます。

/platformまたは/pで全員のプラットフォームが表示されます。

ソースコードは面倒臭いので公開しません。

まあソースコード知りたくなることはないと思うけど。
